# Principles of Data Science - Final Analysis Project

## Project Overview
This repo houses all code and CSVs I used for my FA25 Principles of Data Science course final project. Using Psycopy2 and Pandas, I accessed Acme Grocery Database housed within Amazon Data Servers and parsed through the information.

After determining three actionable insights based on customer demographics and purchasing data, I provided potential marketing and redesign tactics for Acme Grocery based on my analysis.

Technologies Used
- Python
- Jupyter Notebooks
- Pandas
- Psycopg2

## How to run this project - 
1. clone repository -
   ```bash
   git clone https://github.com/ejarredondo/FinalAnalysisProject.git
   cd FinalAnalysisProject
   ```
2. create virtual environment (recommended) -
   ```bash
   python -m venv venv
   source venv/bin/activate    # Mac/Linux
   venv\Scripts\activate       # Windows
   ```
3. install dependencies -
   ```bash
   pip install -r requirements.txt
   ```
4. run the notebook -
   open Juypter Notebook:
   ```bash
   juypter notebook
   ```
   Then, open notebook inside `notebooks/` folder
